介绍
浮点数长度限制在很大程度上影响我进行编程，如圆周率的计算。何况浮点数还有(0.1+0.2==0.3)为false的浮点数狱。为此，我编写了一个Python类。新增了一中数据类型--分数
使用说明

构建 
    - a=fenshu（zi，mu=1）整数变为分数
    - a=fenshu（shu）浮点数变为分数（浮点数精度太高会丢失尾数）
    


使用：
    - 加减乘除均可使用，但与int，float不兼容，需要转换
    - zhi（）方法将其变为float
